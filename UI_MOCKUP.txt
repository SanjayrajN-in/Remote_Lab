STATES AND TRANSITIONS:
────────────────────────

1. Mouse Outside Canvas:
   ├─ Controls: Hidden (opacity: 0)
   ├─ Animation: Fade out (300ms)
   └─ Interaction: Disabled

2. Mouse Entering Canvas:
   ├─ Controls: Visible (opacity: 100)
   ├─ Animation: Fade in (300ms)
   └─ Interaction: Enabled

3. Slider Being Dragged:
   ├─ Value: Updates in real-time
   ├─ Display: Shows percentage (0-100%)
   ├─ API: POST to /logic/config
   └─ Waveform: Updates immediately

4. Mode Button Clicked:
   ├─ Selected: Gets bold text + glow
   ├─ Others: Return to normal state
   ├─ Dropdown: Updates to match
   ├─ API: POST to /logic/config
   └─ Waveform: Shows selected channels only

5. Mouse Leaving Canvas:
   ├─ Controls: Fade out (300ms)
   ├─ Values: Retained for next hover
   └─ State: Unchanged
