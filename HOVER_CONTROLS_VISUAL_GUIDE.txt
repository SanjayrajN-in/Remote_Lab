INTERACTION FLOW
────────────────

User hovers over canvas
        ↓
Fade in (300ms) with opacity animation
        ↓
Controls visible with:
  • CH1 slider (green color-coded)
  • CH2 slider (blue color-coded)
  • Mode selection buttons
        ↓
User adjusts slider
        ↓
Value updates in real-time (0-100%)
        ↓
Sends POST to /logic/config endpoint
        ↓
Amplitude applied to waveform display
        ↓
User hovers away from canvas
        ↓
Fade out (300ms) with opacity animation
